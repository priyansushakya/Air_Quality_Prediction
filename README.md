# AQI Prediction System (Streamlit)

This project predicts Air Quality Index (AQI) using a Random Forest model based on historical air quality data.

## Requirements

- Python 3.8+
- Streamlit
- Pandas
- Scikit-learn

## How to Run the App

1. Open terminal in the project directory
2. Run the following command:

```bash
python -m streamlit run app.py
```
